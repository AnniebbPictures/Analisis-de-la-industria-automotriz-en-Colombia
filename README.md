# Analisis-de-la-industria-automotriz-en-Colombia
A partir de tres fuentes de datos en Excel (Encuesta Mensual de Comercio, EMC de vehículos automotores y Censo Nacional de Población y Vivienda 2018), estudios más profundos sobre el comportamiento de la industria automotriz en Colombia, así como su relación con factores económicos y demográficos.
